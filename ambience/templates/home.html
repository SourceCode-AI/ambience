Welcome to Ambience!

Ambience is a server for aggregating, searching and auditing the results from scans produced by the Aura project. You can use this server to whitelist only audited packages when installing the packages on a client:

  • pip install --index-url {{ base_url }}simple/ package1 package2 ...
  • easy_install --index-url {{ base_url }}simple/ package1 package2 ...
