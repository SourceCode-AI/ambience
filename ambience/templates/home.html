Welcome to Ambience!

Ambience is a server for indexing and aggregating the results of audits of Python packages. Our aim is to continually scan published python packages and to provide the collected data publicly for free.

You can use this server as a PyPI mirror that will allow only audited and whitelisted packages to be installed:

  • pip install --index-url {{ base_url }}simple/ package1 package2 ...
  • easy_install --index-url {{ base_url }}simple/ package1 package2 ...
Powered by Aura @ SourceCode.AI

Auditing and scanning in Ambience is done by the engine called Aura. Developed at SourceCode.AI, this SAST scanner has been designed to audit the whole PyPI package repository and other Python source code on a large scale.
